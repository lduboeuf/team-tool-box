Welcome to Team Tool Box, this is a small project to help you build teams randomly, hope that will help.

All datas are for instance stored in your browser local storage

This is a sample Single Page Application built with several tiny components, more information and source code here: https://github.com/lduboeuf/team-tool-box

Also,demonstrate that for little apps, there is no need to big MV* Frameworks and complex glue code.
